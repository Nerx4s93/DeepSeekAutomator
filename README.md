# DeepSeekAutomator
Автоматизированный C# клиент для взаимодействия с веб-интерфейсом DeepSeek Chat через Selenium. Позволяет управлять режимами чата, отправлять сообщения и получать ответы, используя реальные профили браузера для обхода защиты.

# 🚀 Возможности
- Управление режимами: Переключение между "Обычным" и "Экспертным" режимами.
- Функции нейросети: Включение/выключение "Глубокого мышления" (DeepThink) и "Умного поиска".
- Работа с профилями: Поддержка user-data-dir для сохранения сессий.
- Удобный API: Синхронная отправка сообщений и отслеживание статуса генерации.

# 📦 Установка
Пока, что только скачать и скомпилировать исходный код.

# ⚡ Быстрый старт
Установите пакеты через NuGet:
```
dotnet add package Selenium.WebDriver
dotnet add package Selenium.WebDriver.ChromeDriver
dotnet add package DotNetSeleniumExtras.WaitHelpers
```

Пример использования:
``` C#
using DeepSeekAutomator;

// Инициализация (имя профиля и режим headless)
using var bot = new DeepSeekClient("my_profile", headless: false);

// Настройка режима
bot.EnableDeepThinking();
bot.EnableOnlineSearch();

// Отправка сообщения
bot.SendMessage("Привет! Напиши код на C# для сортировки пузырьком.");

// Ожидание ответа
while (bot.IsGenerating)
{
    Thread.Sleep(200);
}

// Получение текста
string answer = bot.GetLastMessageText();
Console.WriteLine($"Ответ от DeepSeek: {answer}");
```

📚 API
- EnableExpertMode() - Включает модель Expert.
- EnableDeepThinking() - Активирует режим DeepThink.
- EnableOnlineSearch() - Включает поиск в интернете.
- NewChat() - Нажимает кнопку создания нового чата.
- IsGenerating - Свойство (bool), показывающее, пишет ли ИИ ответ в данный момент.
- GetLastMessageText() - Возвращает текст последнего сообщения в чате.
